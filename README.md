Netmiko webapp for backup only with ssh.


1. install requirement package
2. pip install -r requirements.txt
3. create database mysql and import the db.sql
5. edit connection on config/dbconfig.py
6. start the app
7. Login with username admin and password admin1234


Features: 
1. schedul backup 
2. multi device type / brands

tested on juniper,H3C devices
